# 3-Minute Demo Script

## 0:00 - 0:20 Problem

"Most SMB websites lose leads due to hidden UX, mobile, SEO, and performance issues. Agencies take days for audits; SMBs need insight in minutes."

## 0:20 - 0:50 Live URL run

- Enter a URL and click **Audit Now**.
- Mention free/open-source pipeline only: Playwright, Lighthouse, axe-core, Cheerio.

## 0:50 - 1:20 Pipeline progress

- Show progress states while API runs.
- Explain resilience: if a stage fails, partial report is still returned.

## 1:20 - 2:10 Scores + key issues

- Walk through category score cards.
- Highlight top 3 high-priority issues and why they hurt conversions.

## 2:10 - 2:40 Recommendations + business impact

- Show prioritized recommendations (High/Medium/Low).
- Show business impact + estimated lead improvement opportunity.
- Mention 30-day action plan from report.

## 2:40 - 3:00 Close

"This gives SMBs fast, affordable, evidence-based audits without paid APIs. It turns technical findings into action they can execute immediately."
